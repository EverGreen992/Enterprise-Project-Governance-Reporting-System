# Enterprise Project Governance & Reporting System (EPGRS)
## Overview:
The Enterprise Project Governance & Reporting System (EPGRS) is a cloud-based portfolio oversight framework developed using Microsoft 365 tools.
This system demonstrates structured project governance, financial monitoring, risk classification, and real-time collaborative documentation within a controlled enterprise environment.
## Objectives:
- Establish structured project tracking
- Monitor budget vs actual spend
- Classify and prioritize risk exposure
- Enable executive dashboard visibility
- Demonstrate real-time collaboration
## Tools Used
- Microsoft Excel (Project Tracking & Dashboard)
- Microsoft Word (Governance Documentation)
- Microsoft OneDrive (Cloud Collaboration)
## Key Features
## 1. Project Portfolio Tracking
- Status classification
- Budget variance calculation
- Risk level monitoring
## 2. Executive Dashboard
- Total Budget & Spend
- Variance Monitoring
- Risk Distribution
- Project Health Overview
## 3. Governance Framework
- Structured reporting methodology
- Risk scoring model
- Accountability mapping
## 4. Collaboration Demonstration
- Real-time co-authoring
- Comment resolution workflow
- Track Changes review
